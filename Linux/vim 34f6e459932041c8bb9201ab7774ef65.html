整体命令

Linux系统使用Shell整体命令。我们可以通过终端控制台来输入命令，由shell进行解释并最终交给内核执行。

ls 命令用于显示系统所有的文件内容

Shell命令的实现

在Linux中，Shell的实现使用的是sh，即Bash Shell解释器。到后来为了简化，发布许多发行版如Debian、Ubuntu。CentOS默认使用的是bash。即sh链接到的是bash

-l 命令表示将下列信息每一行完整的列举出来，可以加上一个路径如/bin/ ，| 代表管道，grep XXX 表示搜索所有带有“XXX”的文件。完整命令:

ls -l /bin/ | grep sh

man命令

man 指令可以提供帮助文档，解释其他的命令。针对的是外部命令。对于内置命令，可以在man 后面加-f 。

man ls
man -f cd

help命令

help命令主要针对Bash内置命令提供帮助文档，缺点是纯英文。对于外部命令，可以使用--help ，其汉化效果优于help 。

help cd
ls --help

一般命令的选项中，长项（或者说全称）一般用“--”；而同一命令的短项（或者说简称）一般用“-”。

文件目录类

pwd 全称print working directory打印工作目录，显示当前工作目录的绝对路径、ls 列举目录内容

、cd change directory切换目录。跳转到桌面：cd /root/桌面

切换用户：su

ls -a 所有文件。. 是当前目录文件，.. 是上一级目录的文件。以. 开头的文件均为隐藏文件。ls -l 则是把属性文件全部列出，简写ll。

mkdir 用于创建文件夹。参数-p 用于多级创建。mkdir -p a/b/c 。rmdir 用于删除文件夹。

touch 用于创建一个空文件。可以加路径指定创建。vim进入编辑后，q退出文件会消失，wq 则是保留空文件。

cp 复制。语法为 cp [选项] source dest 将source文件复制到 目标文件dest下。此时dest文件的内容会提示是否覆盖。如果想强制覆盖dest文件不提示可以将cp 写成\cp 。其参数为：-r 递归复制整个文件夹。

rm 删除。语法为 rm [选项] file 。参数： r递归删除文件夹里的每一个文件 f强制不提示删除 /根目录 *通配符。删库跑路究极指令：rm -rf /*

mv 移动。与复制语法相同。

cat 查看。参数：-n 显示所有的行号，包括空行。翻页空格，回车一行一行显示，b 向后翻页，f 向前翻页。= 可以显示当前行号。:+f 显示什么文件多少行。

less 分屏显示内容。其功能与more 指令相似，用于打开一些很大的日志文件。shift+g 可以直接跳转到最后一页，g跳转到第一页。搜索：/ 。n 搜索关键词下一个，shift+n 反之。

head tail 显示文件的开头与结尾，参数-n 指定行数。tail -f 用于查看追踪该文档的所有更新，部署项目时实时查看日志就很方便。但是，如果使用vim编辑文件后，文件不会实时更新，这是因为使用vim编辑后文件的索引号已经发生了变化，不是原来的文件了，就无法继续监听

echo 用于输出内容。

tail 的监听使用方法

先tail -f info ，输出的文件的具体信息，此时显示内容并没有退出，我们再打开一个终端，在此终端中写echo hello 追加内容，显示的内容就会实时更新。如果使用vim追加内容，监听的参数需要改成-F 。

ln 软连接，类似于Windows的快捷方式，有自己的数据块，存放文件的路径。语法：ln -s [路径][链接名] ，可以通过pwd显示路径，如果需要显示实际的路径而不包括软连接名字，需要使用pwd -P 指令。

删除软连接rm -f 文件名时，不会影响真实文件，但是当执行rm -rf 文件名/ 时，会将真实目录下的子文件全部删除。

history 用于查看历史执行记录，后面可以接数字表示显示的行数，参数-c 表示清空历史记录
