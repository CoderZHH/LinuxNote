Shell

变量定义

unset 变量：撤销变量

readonly 变量=值 ：静态变量

export 变量名=变量值：将shell变量输出为环境变量

source 配置文件：让修改后的配置信息生效

echo $变量名：查询环境变量的值

A=`date`：`执行完了赋值`，等价于$(date)

位置参数（可以理解成传参）

$n ：n为数字，$0代表命令本身，$1-$9代表第一到第九个参数，十以上的参数，十以上的参数需要用大括号包含，如${10}

$*：所有的参数，$*把所有的参数看成一个整体

$@：所有的参数，不过$@把每个参数区分对待

$#：所有参数的个数

!#/bin/bash
echo "1=$1 2=$2"
echo "所有的参数=$*"
echo "$@"
echo "参数的个数=$#"
./xxxx.sh 100 200

运算符

基本语法

  1. "$((运算式))" 或 "$[运算式]" 或者 expr m + n
  1. 注意expr运算符间要有空格
  1. expr m - n
  1. expr \\*, /, % 乘，除，取余
#!/bin/bash
# 案例1：计算 (2+3) X4的值
# 使用第一种方式
RES1=$(( (2+3)*4 ))
echo "res1=$RES1"
# 使用第二种方式，推荐使用
RES2=$[ (2+3)*4 ]
echo "res2=$RES2"
# 使用第三种方式 expr
TEMP=`expr 2 + 3`
RES4=`expr $TEMP \* 4`
echo "temp=$TEMP"
echo "res4=$RES4"

# 案例2：请求出命令行的两个参数[整数]的和 20 50
SUM=$[$1+$2]
echo "sum=$SUM"

Read

read命令有许多选项和参数，可以用于不同的需求。以下是常用的read选项和参数的详细介绍：

常用选项

  • a array: 将读取的单词分配给数组变量。
  • d delim: 使用delim作为分隔符而不是默认的换行符。
  • e: 使用readline库来获得输入，这允许使用编辑功能（例如，箭头键）。
  • n nchars: 读取指定数量的字符而不是整行。
  • p prompt: 在读取输入前显示提示信息。
  • r: 忽略反斜杠的特殊含义，即不把反斜杠作为转义字符。
  • s: 隐藏输入内容，通常用于输入密码。
  • t timeout: 等待指定的时间（秒）后，如果没有输入，则返回非零状态。
  • u fd: 从文件描述符fd读取输入。

示例

示例 1: 使用数组

#!/bin/bash

echo "Enter a list of items (space-separated):"
read -a items
echo "You entered: ${items[@]}"

示例 2: 使用自定义分隔符

#!/bin/bash

echo "Enter a string with colons:"
read -d ':' input
echo "You entered: $input"

示例 3: 使用readline库

#!/bin/bash

read -e -p "Enter your command: " command
echo "You entered: $command"

示例 4: 读取指定数量的字符

#!/bin/bash

echo "Enter a character:"
read -n 1 char
echo "You entered: $char"

示例 5: 隐藏输入内容

#!/bin/bash

read -sp "Enter your password: " password
echo
echo "Password entered."

示例 6: 设置超时时间

#!/bin/bash

if read -t 5 -p "Enter something within 5 seconds: " input
then
    echo "You entered: $input"
else
    echo "Timed out!"
fi

数组

  • ${array[@]}：引用数组中的所有元素。
  • ${array[i]}：引用数组中的第 i 个元素（索引从0开始）。
  • ${#array[@]}：获取数组中元素的个数。
