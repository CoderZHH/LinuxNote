文件目录指令

mkdir ：创建一个目录

-p 创建多层级目录

rmdir：删除空目录

-rf ：强制删除

touch：创建空文件

cp [选项] sourse dest

-r ：递归复制整个文件

\cp ：强制覆盖，不提示

rm [选项] sourse：删除

-r：递归删除整个文件夹

-f：强制删除不提示

mv oldNameFile newNameFile ：重命名

mv /temp/movefile /targetFolder：移动文件

mv /temp/movefile /targetFolder/newNameFile ：移动并重新命名文件

cat [选项] 要查看的文件

-n ：显示行号

echo $HOSTNAME

head -n 5 文件

tail -n 5 文件

tail -f mytext.text：监听这个文件有无更新

echo "hello" > /home/mytext..text：（可以是另一个用户的操作）通过echo将文字重定向到文件中

tail的指令输出hello

>指令 和 >>指令

>输出重定向和 >>追加

1)ls -l >文件：(功能描述:列表的内容写入文件a.txt中(覆盖写))

2)ls -al >>文件：(功能描述:列表的内容追加到文件a.txt的末尾)

3)cat 文件1 >文件2：(功能描述:将文件1的内容覆盖到文件2)

4)echo"内容">>文件

ln -s [原文件或目录] [软连接名]

ln -s /root /home/myroot

在home下创建一个myroot软连接指向root文件夹

find [搜索范围] [选项] :在搜索范围内查找对应文件

- name <查询方式>

- user <用户名>

- size <文件大小> （+n大于 -n小于 n等于，单位k,M,G）

find /home -name hello.txt

find /opt -user zhh

find / -size +200M

updatedb ：创建文件位置的数据库，供locate直接找

locate [文件名]

grep [选项] 查找内容 源文件

-n 显示行号

-i 忽略字母大小写

cat /home/hello.txt | grep “yes”

grep -n “yes” /home/hello.txt

gzipe压缩文件

gunzip解压文件

zip ：压缩

-r ：递归压缩

zip -r myzip.zip /home：压缩home文件夹

unzip ：解压

-d <目录> ：指定解压目录

unzip -d /opt/tmp /home/myzip.zip：解压到/opt/tmp

tar [选项] XXX.tar.gz

-c：产生.tar打包文件

-v：显示详情信息

-f：指定压缩后的文件名

-z：打包同时压缩

-x：解包.tar文件

tar -zcvf pc.tar.gz /home/pig.txt /home/cat.txt：多个文件打包压缩中间用空格隔开

tar -zxvf pc.tar.gz ：解压到当前目录

tar -zxvf /home/myhome.tar.gz -C /opt/tmp2

-C 理解成切换目录

-z先压缩成.gz文件，c再打包成.tar包文件。

压缩和解压都是要z的，区别就在于：

有x解包的时候z对应的就是解压的意思，

有c打包的时候z就是压缩的意思
