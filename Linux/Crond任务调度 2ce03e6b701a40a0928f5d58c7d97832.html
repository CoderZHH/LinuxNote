Crond任务调度

-e：编辑crontab定时任务

-l：查询crontab任务

-r：删除当前用户所有的crontab任务

service crond restart：重启任务调度

* * * * * command to be executed
- - - - -
| | | | |
| | | | +---- 星期几 (0 - 7) (Sunday = 0 or 7)
| | | +------ 月份 (1 - 12)
| | +-------- 月份中的第几天 (1 - 31)
| +---------- 小时 (0 - 23)
+------------ 分钟 (0 - 59)

每天晚上11：30备份数据库

30 23 * * * /path/to/backup/script.sh

每天周一早上8：00发送报告

0 8 * * 1 /path/to/report/script.sh

每5分钟执行一次清理临时文件任务

*/5 * * * * /path/to/cleanup/script.sh

  • */：表示步长，用于定义每隔多少单位执行一次

0 8,12,16 * * * 每天的8点0分，12点0分，16点0分都执行一次

  • ,：表示不连续的时间

0 5 * * 1-6 每周一到周六的凌晨5点0分

  • -：表示连续时间的范围
